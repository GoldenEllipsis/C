    爱因斯坦数学题。爱因斯坦曾出过这样一道数学题：有一条长阶梯，若每步跨2阶，
最后剩下1阶；若每步跨3阶，最后剩下2阶；若每步跨5阶，最后剩下4阶；若每步跨6阶，
最后剩下5阶；只有每步跨7阶，最后才正好1阶不剩。请问，这条阶梯共有多少阶？

#include <stdio.h>
int main()
{
    int  x = 7, find = 0;
    while (!find)
    {
        if (x%2==1 && x%3==2 && x%5==4 && x%6==5 && x%7==0)
        {
                printf("x = %d\n", x);
                find = 1;   
        }
        x++;
    }
    return 0;
}
